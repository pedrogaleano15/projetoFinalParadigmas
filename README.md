# Projeto Final — Paradigmas de Programação

Projeto integrador desenvolvido como trabalho final da disciplina de **Paradigmas de Programação** na graduação em Engenharia da Computação (UCDB). O projeto explora na prática os principais paradigmas de desenvolvimento de software.

---

## Sobre o projeto

O objetivo foi implementar uma solução que demonstre a aplicação de diferentes **paradigmas de programação**, evidenciando como o mesmo problema pode ser abordado de formas distintas dependendo do paradigma adotado.

---

## Paradigmas abordados

### Orientado a Objetos (POO)
Modelagem de entidades do mundo real através de **classes, objetos, herança, encapsulamento e polimorfismo**. O Java foi utilizado como linguagem principal por sua forte tipagem e suporte nativo ao paradigma.

**Conceitos aplicados:**
- Abstração e encapsulamento de dados
- Herança e sobrescrita de métodos
- Polimorfismo e interfaces
- Composição de objetos

---

### Paradigma Funcional
Aplicação de conceitos funcionais como **funções de alta ordem, imutabilidade e expressões lambda** disponíveis no Java moderno.

**Conceitos aplicados:**
- Lambdas e Streams da API Java
- Funções puras e imutabilidade
- Processamento declarativo de coleções
